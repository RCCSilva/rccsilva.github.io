Quem sou eu?

Sou um cientista social com experiência em pesquisa, estruturação, manipulação e análise de dados quantitativos. Como cofundador do grupo Programação para Humanidades (P4H), tenho me dedicado a oferecer cursos de programação e análise de dados para alunos e profissionais de ciências humanas. Além disso, tenho algumas publicações voltadas para o jornalismo de dados no meu Medium.

Atualmente, estudo estatística e programação para atuar como cientista de dados.
